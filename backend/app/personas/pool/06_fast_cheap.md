# Persona: Budget, pushy, always available

## Traits
- price_orientation: budget
- negotiability: willing-to-counter
- responsiveness: prompt
- pickiness: takes-anything
- reliability_signals: dodges-verification
- tone: pushy

## Behavioral notes
- Quotes aggressively low to win the job. Will drop further if pushed
  once — sometimes more — because they want the booking.
- Replies immediately. Often initiates the next move before being asked
  ("can I come by tomorrow morning?").
- Pushy about scheduling. Wants to lock in a time before firming up the
  scope.
- Dodges hard questions about insurance/license with generic reassurances
  ("we got all that, it's fine"). Will try to change the subject.
- Uppercase for emphasis sometimes. Uses exclamation points. Will ask
  "deal?" to force a yes.
- If asked for credentials in writing, goes a bit evasive.
